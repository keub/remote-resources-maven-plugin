remote-resources:copy

Full name:

com.github.keub:remote-resources-maven-plugin:1.0:copy

Description:

Copy resources for the main source code to the main output directory. Always uses the project.build.resources element to specify the resources to copy.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Required Parameters

Name Type Since Description
outputDirectory File - The output directory into which to copy the resources.
resources Set - Set of resource

Parameter Details

outputDirectory:

The output directory into which to copy the resources.
  • Type: java.io.File
  • Required: Yes

resources:

Set of resource
  • Type: java.util.Set
  • Required: Yes
